# Présentation du projet RimLink

## 1. Introduction

RimLink est une application mobile de réseau professionnel développée avec Flutter. Elle permet de connecter les utilisateurs et les entreprises dans un espace professionnel simple et interactif.

## 2. Objectif du projet

L’objectif de RimLink est de faciliter la création de profils professionnels, le partage de publications, la gestion du réseau professionnel et la consultation des offres d’emploi.

## 3. Fonctionnalités principales

- Inscription et connexion
- Gestion du profil professionnel
- Ajout des expériences professionnelles
- Ajout des formations
- Création et affichage des publications
- Likes, commentaires et interactions
- Recherche et connexion entre utilisateurs
- Consultation et publication des offres d’emploi
- Gestion des informations de contact

## 4. Technologies utilisées

- Flutter
- Dart
- Supabase
- PostgreSQL
- GitHub

## 5. Architecture du projet

Le projet est organisé par fonctionnalités afin de séparer l’interface utilisateur, la logique métier et les données.

Les principales parties sont :

- Authentification
- Profil
- Publications
- Réseau professionnel
- Offres d’emploi
- Services Supabase

## 6. Démonstration

La démonstration présente la page d’inscription, la connexion, la navigation principale, les profils, les publications et les offres d’emploi.

## 7. Conclusion

RimLink applique les notions étudiées en développement mobile : interface interactive, navigation, gestion des données, authentification, backend, base de données et organisation du code.
